PecaStarter5
============

概要
--------
Peercastでの配信を補助するツールです。

動作環境
--------
Windows XP SP3以上
.NET Framework 4 Client Profile
Peercast-IM または Peercast Station

使い方
--------
Peercastが起動され、Windows Media EncoderまたはKTE等のエンコードが開始されていることを確認します。
情報を入力し、配信開始ボタンを押すとリレーが開始されます。
